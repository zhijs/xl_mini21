### mini 项目脚手架


### 1.目录结构
```
|--src           源代码目录
|  |--api        接口请求相关逻辑  
|  |--assets     资源文件目录
|  |--components 可复用组件目录
|  |--pages      页面组件目录
|  |--store      redux相关目录
|  |--util      公用工具函数目录
```

### 2.开发环境搭建
https://www.jianshu.com/p/12674aa384a1


3.clone 远程代码,并安装依赖
```javascript
git clone https://github.com/zhijs/xl_mini21.git
npm install
```


### 运行
```
react-native run-ios
react-native run-android

react-devtools 开启调试器
react-native log android 输出日志 console.log
```
